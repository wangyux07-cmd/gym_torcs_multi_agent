RaceYourCode TORCS AI Driver

A Python-based TORCS racing project that combines a racing controller, reinforcement learning experiments, telemetry, speed profiles, and a web race control interface.

  • Simulator: TORCS with SCR server
  • Language: Python
  • Learning tools: Gymnasium, Stable-Baselines3 PPO and SAC
  • Interface: FastAPI backend and static web frontend

RaceYourCode is a solo TORCS AI racing project.
